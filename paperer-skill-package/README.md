# paperer-skill-package

This is the minimal distributable `Paperer` skill package.

- Start with [SKILL_PACKAGE.md](SKILL_PACKAGE.md)
- The default entry skill is `skills/paper-package-runner/SKILL.md`
- This package is self-contained for normal Paperer skill usage
- Do not use `scripts/rebuild_*.py` unless you intentionally need repo-maintainer tooling
